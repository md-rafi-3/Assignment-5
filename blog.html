Question-1: What are the different ways to select an element in the DOM?

DOM থেকে এলিমেন্ট সিলেক্ট করার বিভিন্ন উপায়:

getElementById()

এটি নির্দিষ্ট ID দিয়ে একটি এলিমেন্ট কে সিলেক্ট করে

getElementByClassName

এটি দিয়ে একই ক্লাসের সব এলিমেন্ট কে সিলেক্ট করা যায়

getElementByTagName

এটি দিয়ে নির্দিষ্ট কোন Tag এর নাম ব্যবহার করে সব এলিমেন্ট কে সিলেক্ট করা যায়

getElementByName()

এটি দিয়ে name Attributeব্যবহার করে এলিমেন্ট সিলেক্ট করা যায়

querySelector()

এটি CSS সিলেক্টর ব্যবহার করে প্রথম মিলে যাওয়া এলিমেন্ট সিলেক্ট করে

querySelectorAll()

এটি CSS সিলেক্টর ব্যবহার করে সব মিল থাকা এলিমেন্ট সিলেক্ট করে

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML,innerText এবং textContent এর মধ্যে পার্থক্য হলো:

innerHTML :এটি ব্যবহার করে একটি এলিমেন্টের সম্পূর্ণ HTML কোড পরিবর্তন করা যায়

innerText : এটি ব্যবহার করে একটি এলিমেন্টের দৃশ্যমান টেস্ট এর এক্সেস নেওয়া যায় যা দিয়ে দৃশ্যমান text পরিবর্তন করা যায় এটি হিডেন টেস্ট এর এক্সেস নিতে পারেনা

textContent :এর কাজ innerText মতোই কিন্তু এটি দৃশ্যমান text পাশাপাশি hidden text এর এক্সেস নিতে পারে
